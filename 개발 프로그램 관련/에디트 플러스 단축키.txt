1. 문서 생성과 파일관련 단축키
 

¤ 새 문서 작성 ( Ctrl + N ) : 보통 텍스트 문서를 새로 작성

¤ 새 HTML 페이지 ( Ctrl + Shift + N) : HTML 페이지를 새로 작성

¤ 열기 ( Ctrl + O ) : 파일열기 

¤ 저장 ( Ctrl + S ) : 현재 열린 문서를 저장 




2. 커서 이동과 선택관련 단축키
 


 

¤ 이전 탭 ( Shift + Tab ) : 커서를 이전 탭 위치로 이동

¤ 문서 끝 ( Ctrl + End ) : 문서의 끝으로 이동 



[EditPlus] 에디트플러스 단축키 모음과 설정하기

 

 

¤ 문서 처음 ( Ctrl + Home ) : 문서의 처음으로 이동  

¤ 줄 끝 ( End ) : 커서를 현재 줄의 끝으로 이동 

¤ 줄 끝 선택 ( Shift + End ) : 선택 부분을 현재 줄의 끝까지 확장



[EditPlus] 에디트플러스 단축키 모음과 설정하기

 

 

¤ 줄 처음 ( Home ) : 커서를 현재 줄의 처음으로 이동 

¤ 줄 처음 선택 ( Shift + Home ) : 선택 부분을 현재 줄의 처음까지 확장 

¤ 쪽 아래 (  Page Down ) : 커서를 한 쪽 아래로 이동

¤ 쪽 위 ( Page Up ) : 커서를 한 쪽 위로 이동

¤ 쪽 위 선택 ( Shift + Page + Up ) : 선택 부분을 한 쪽 위로 확장

 



3. 들여쓰기와 실행관련 단축키
 


들여쓰기 줄임 ( Ctrl + Shift + I  ) : 현재 줄이나 선택 부분의 들여쓰기를 줄임

줄 끝까지 지우기 ( Ctrl + Shift + Delete  ) : 현재 줄을 끝까지 지움 

들여쓰기 늘임 ( Ctrl + I ) : 현재 줄이나 선택 부분의 들여쓰기를 늘임 

다시 실행 ( Ctrl + Y ) : 이전에 취소한 동작 다시 수행

실행 취소 ( Ctrl + Z ) : 마지막으로 한 동작 취소



4. 찾기관련 단축키
 


¤ 코드 생략 사용 ( Ctrl + Shift + F ) : 코드 줄임기능 켜거나 끔 


¤ 찾기( Ctrl + F ) : 지정한 문자열을 찾음  

¤ 다음찾기 ( F3 ) : 다음 일치하는 단어나 문장을 찾음  

¤ 찾아 바꾸기 ( Ctrl + H ) : 지정한 문자열을 다른 문자열로 바꿈 

¤ 줄 찾아가기 ( Ctrl + G ) : 팝업창에 설정한 줄번호로 이동

  


5. 단축키 설정 
 


현재 사용하고 있는 단축키나 단축키가 설정되어 있지 않은 기능들에 대해서 셋팅이 가능합니다. 
[도구]메뉴의 [기본설정] 을 클릭해서 대화상자를 띄웁니다. 


[기본 설정] 대화상자의 항목 리스트에서 [키보드] 를 선택합니다. 
단축키 정보를 수정하거나 추가할수 있는 곳입니다. 

번호 순서대로 [유형] 을 선택하고 유형 내부에 명령을 고릅니다. 
단축키를 설정하고 싶은 명령을 고른후 [새 단축키 누리기] 란에 키보드를 누릅니다. 
마지막으로 설정한 단축키를 적용하기 위해 [할당] 버튼을 클릭하고 저장하시면 끝납니다.
