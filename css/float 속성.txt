float(부유) 속성은 레이아웃을 잡을 때 사용하는 속성 정도로 이해하기 쉬운데
float 속성이 처음부터 레이아웃용 속성으로 만들어진 것이 아니라
과거 웹 디자인에서는 그림 주위를 흐르는 텍스트 속성으로 html<img> 요소에
속한 align 표현 속성을 이용했는데, 현대 웹 디자인으로 넘어오면서 html에
있던 표현 요소가 대부분 css로 대체되었습니다.
HTML align 속성을 대체할 CSS 속성으로 float 속성이 만들어진 것입니다.

float 속성을 이용하면 다음과 같이 그림 주위를 둘러싼(흔히'그림 주위를
흐른다'라고 표현합니다) 텍스트를 만들수 있씁니다 그렇다고 해서
float 속성이 <img>요소에만 적용되는 것은 아닙니다.
float 속성은 align 속성과 달리 문서를 구성하는 모든 요서에 적용할 수
있습니다. 예를 들어 <div> 요소나 <p>요소에도 적용할 수 있습니다.

float 속성은 적용된 요소를 어느 위치에 띄울 것인지를 정합니다.
상대적으로 적용된 요소가 어느 위치에 띄워질 경우,
그곳에 위치하고 있던 다른 요소는 옆으로 밀려나게 됩니다.
이러한 특징은 앞서 살펴본 absolute(절대위치)와 비슷해 보입니다.
하지만 float는 absolute와 확실히 다른 몇 가지 특징이 있습니다.
다른 점을 세가지 정도 정리하면 

1.absolute 와 float는 위치를 지정하는 방법이 다르다!

absolute는 부모 요소 박스의 각 모서리가 기준점이 됩니다.
따라서 네 방향을 기준으로 이동합니다. float는 left와 right속성 값에 따라
부모 요소 박스의 좌상단과 우상단 모서리만 기준점이 됩니다. absolute처럼
좌하단이나 우하단 모서리는 기준짐이 될수 없습니다.

2.absolute 속성이 부여된 요소는 텍스트가 둘러싸지 못하지만 
float 속성이 부여된 요소는 텍스트가 둘러 쌀 수 있다!

A요소 와 B요소가 수평으로 나란히 있다고 가정해보겠습니다.
A 요소에 absolute 속성을 설정하면 바로 아래에 있던 B요소가
A요소가 있던 자리로 올라옵니다. 부유(float)도 마찬가지로 B요소가
A요소가 있던 자리도 올라옵니다. 차이점은 B요소가 텍스트 요소일때 
나타납니다. B요소가 텍스트 요소 일 경우 A요소에 absoulte 속성이 설정되면
B 텍스트 요소는 뒤로 숨겨지는데 반해,float가 설정되면 B텍스트 요소는
A요소를 둘러싸게 됩니다(글자가 요소 옆으로 흐릅니다.)

3.float는 영향력을 제어할 수 있는데 반해 absolute는 영향력을 제어 할수 없다.

float와 absolute는 모두 기본 흐름(normal flow)을 벗어나는 특징이 있기
때문에 아래에 놓여 있던 요소가 부여하거나 옮겨지게 됩니다.
이것은 곧 float와 absolute가 아래에 위치한 요소에 영향을 끼친다고 정리
할 수 있습니다. 다만 두속성은 차이가 있습니다. 
레이아웃 속성인 absolute는 아래에 위치한 요소와 곂치지 않게 배치하는
효율적인 방법이 없습니다. absolute된 요소의 높이 값만큼 뒤에 위치한 
요소의 위쪽 마진(margin-top) 값을 설정할 수 있지만, absolute된 요소의
높이 값이 바뀔 경우 두 요소는 겹쳐지는 문제가 발생합니다.
반면 float는 뒤에 위치한 요소에 clear 속성을 적용하면 영향력을 벗어날
수 있습니다. 이런 차이점 때문에 레이아웃 속성이 아닌 float가 레이아웃
속성인 absolute보다 레이아웃 작업을 할 때 더 많이 활용되는 것입니다.
