[GAC(Global Assembly Cache)]

응용 프로그램에서 공유하도록 설치되는 어셈블리를 저장하는 전체 컴퓨터에 사용되는 캐시.
공통 언어 런타임(CLR)이 설치된 컴퓨터에는 응용 프로그램에서 공유하도록 설치된 어셈블리를 저장하는
범용 어셈블리 캐시(GAC) 기능이 있다. 닷넷에는 캐시와 관련한 두가지 도구가 있는데,
그래픽 사용자 인터페이스를 사용해 캐시를 이용하는 원도즈 셀 확장과
GAC 두구라 부르는 명령 줄 도구이다.

========================================================================

저장 위치 : DLL이 저장되어 있는곳
%windir%\Microsoft.NET\assembly  또는 %windir%\assembly


GAC는 여러 애플리케이션이 라이브러리를 공유할 때만 사용하는 것이 좋음
