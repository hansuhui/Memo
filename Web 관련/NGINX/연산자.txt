[연산자]

= : 같다

!= : 다르다

~ : 정규표현식 패턴 매칭. 아래는 .php로 끝나는 ($) 경로로 접근하는 요청에 대해서 error_log를 
출력하는 예제다.

location ~ /.php$ {
    error_log /var/log/opentutorials.org.php.error debug;
}


!~ : ~의 반대

~* : 대소문자를 구분하지 않는 정규표현식 패턴 매칭

!~* : ~* 의 반대

-f : 파일이 존재하는지 여부를 테스트한다.

!-f  : -f의 반대

-d : 디렉토리가 존재하는지 테스트

!-d : -d의 반대

-e : 파일, 디렉토리 , 심볼릭 링크가 존재하는지 여부를 테스트 한다.

!-e : -e의 반대

-x : 파일이 존재하고 실행 가능한지를 테스트

!-x : -x 의 반대
