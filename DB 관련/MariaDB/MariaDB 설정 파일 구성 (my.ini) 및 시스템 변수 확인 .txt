[설정 파일 구성 (my.ini) 및 시스템 변수 확인]

MySQL은 단 하나의 설정 파일만 사용한다. 원도우에서는 my.ini(리눅스에서는 my.cnf) 이름을 사용한다.
이 파일은 수정 할 수 없으며 MySQL이 시작 될 때 설정 파일을 참고 한다.

MySQL에서는 설정 파일이 여러 개 존재 할 수 있는데 서버가 폴더를 읽으면서 처음 만나는 설정 파일을 
사용한다. 여러 개의 설정 파일이 있을 때 어느 파일이 사용되었는지 확인이 필요하다면
다음 명령어를 실행하여 확인 할 수 있다.

mysql -help

MySQL 설정파일 하나에는 여러 개의 설정 그룹을 저장 할 수 있다. 기본적으로 그룹명은
바이너리 프로그램 (mysplid , clinet 등) 이름을 사용한다.

일반적으로 각 그룹을 사용하는 프로그램은 성격이 다르며 각 프로그램이 필요로 하는 설정이
상이 하지만 중복으로 사용되는 정보 (port , socket 등) 각 그룹에 각각 설정해 주어야 한다.

MySQL 서버는 서비스를 시작하면서 설정 파일의 내용을 별도로 저장하는데 이렇게 저장된 것을
시스템 변수라고 한다. 시스템 변수를 확인 하는 방법은 MySQL에 접속하여 다음 명령을 실행한다.

show global variables;

시스템 변수는 글로벌 변소와 세션 변수를 있다. 글러벌 변수는 서버 인스턴스에서 전체적으로 
영향을 미치는 변수이며 세션 변수는 MySQL에 접속할 때 기본적으로 부여하는 옵션의 기본값을 제어할 때
사용된다. 글로벌 및 세션 변수 모두 포함된 경우 both로 표시된다. 각 변수에 대한 설명은 공식 
메뉴얼을 참고 한다.

시스템 변수 사용 : https://dev.mysql.com/doc/refman/5.5/en/using-system-variables.htm
