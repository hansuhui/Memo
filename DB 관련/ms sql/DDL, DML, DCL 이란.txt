


-DDL
내용 : Data Definition Language  (데이터 정의어)
종류 :  CREATE,ALTER,DROP,TRUNCATE


-DML
내용 : Data Manipulation Language(데이터 조작어)
종류 : SELECT,INSERT,UPDATE,DELETE

-DCL
내용 : Data Control Language(데이터 제어어)
종류 : GRANT,REVOKE





 - DDL 이란?
?   데이터베이스를 정의하는 언어를 말하며, 데이터를 생성하거나 수정하거나 삭제하는 등
   데이터의 전체 골격을 결정하는 역활의 언어를 말합니다
 

CREATE :데이터베이스, 테이블등을 생성하는 역활을 합니다.

ALTER : 테이블을 수정하는 역활을 합니다.
 
DROP : 데이터베이스, 테이블을 삭제하는 역활을 합니다.
  
TRUNCATE : 테이블을 초기화 시키는 역활을 합니다.


 
 - DML 이란?
   정의된 데이터베이스에 입력된 레코드를 조회하거나 수정하거나 삭제하는 등의 역활을 하는 언어를 말합니다.

 
SELECT : 데이터를 조회하는 역활을 합니다.
 
INSERT : 데이터를 삽입하는 역활을 합니다.

UPDATE : 데이터를 수정하는 역활을 합니다.

DELETE : 데이터를 삭제하는 역활을 합니다.


- DCL 이란?
?   데이터베이스에 접근하거나 객체에 권한을 주는등의 역활을 하는 언어를 말합니다.
