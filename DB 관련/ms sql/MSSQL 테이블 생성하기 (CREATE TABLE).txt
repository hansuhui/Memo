- MSSQL 테이블 생성방법
 
   CREATE TABLE [테이블명] (
       [컬럼명] [데이터형식] [제약조건]
   )
?
 
 [참고사항] 자주사용하는 데이터 형식
데이터형식 / 설명
 int / 정수데이터를 사용하는 데이터형식 (범위 : -2,147,483,648 ~ 2,147,483,647)
 bigint / 정수데이터를 사용하는 데이터형식 (범위 : -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807)
 float / 부동 소수점 숫자 데이터에 사용하는 근사 숫자 데이터 형식
 datetime / 날짜 데이터를 사용하는 데이터형식
 char / 고정 길이 비유니코드 문자열 데이터형식
 varchar / 가변 길이의 비유니코드 문자열 데이터형식
 text / 최대 문자열 길이가 2,147,483,647인 비유니코드 가변 길이 데이터형식
 
 [참고사항] 자주사용하는 제약조건
제약조건 /설명
 primary key / 테이블의 각 행을 고유하게 식별하는 값을 가진 열 또는 열 조합을 생성하는 제약조건
 foreign key / 두 테이블의 데이터 간 연결을 설정하고 강제 적용하는 데 사용되는 열을 생성하는 제약조건
 unique / 특정 열에 중복 값이 입력되지 않도록 하는 제약조건
 default / Null값이 들어오는 경우 설정한 값을 대신 기록하는 제약조건
 null / not null / Null값을 허용하거나 허용하지 않는 제약조건
 identity / 컬럼에 자동으로 증가값을 부여하는 제약조건
 check / 열에서 허용되는 값을 제한하는 제약조건
